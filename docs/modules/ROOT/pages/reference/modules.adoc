= Module Overview
:description: Guide to Hylograph Selection modules
:keywords: modules, API, reference

This page provides an overview of the module structure. For complete API documentation, see https://pursuit.purescript.org/packages/purescript-hylograph-selection[Pursuit^].

== Core Modules

`Hylograph`::
Re-exports the essential API. Start here for most use cases.

`Hylograph.AST`::
Tree building: `elem`, `named`, `withChild`, `joinData`, etc.

`Hylograph.Expr.Friendly`::
Attribute DSL: `num`, `text`, `width`, `cx`, `fill`, etc.

`Hylograph.Render`::
DOM rendering: `runD3`, `select`, `renderTree`.

== Interpreters

`Hylograph.Interpreter.D3`::
The primary DOM renderer.

`Hylograph.Interpreter.English`::
Human-readable descriptions.

`Hylograph.Interpreter.Mermaid`::
AST structure diagrams.

`Hylograph.Interpreter.MetaAST`::
Untyped AST conversion.

== Internal Modules

These are implementation details but useful for advanced use:

`Hylograph.Internal.Behavior.Types`::
Behaviors: `Drag`, `Zoom`, `onCoordinatedHighlight`.

`Hylograph.Internal.Selection.*`::
Low-level selection operations.

`Hylograph.Internal.Transition.*`::
Animation primitives.

== Data Modules

`Hylograph.Data.Node`::
`SimulationNode` for force layouts.

`Hylograph.Data.Tree`::
Tree data structures.

`Hylograph.Data.Graph`::
Graph structures and algorithms.

== Expression System

`Hylograph.Expr.Expr`::
Core expression types.

`Hylograph.Expr.Attr`::
Attribute constructors.

`Hylograph.Expr.Interpreter.*`::
Expression interpreters (Eval, SVG, CodeGen).

== Scale Module

`Hylograph.Scale`::
D3 scale wrappers: `linear`, `ordinal`, `band`, color schemes.

== Import Patterns

=== Basic Visualization

[source,purescript]
----
import Hylograph.AST as A
import Hylograph.AST (ElementType(..))
import Hylograph.Expr.Friendly (num, text, width, height, cx, cy, r, fill)
import Hylograph.Render (runD3, select, renderTree)
----

=== With Behaviors

[source,purescript]
----
import Hylograph.Internal.Behavior.Types
  (Behavior(..), defaultDrag, defaultZoom, onCoordinatedHighlight)
----

=== With Scales

[source,purescript]
----
import Hylograph.Scale (linear, domain, range, applyScale)
----
